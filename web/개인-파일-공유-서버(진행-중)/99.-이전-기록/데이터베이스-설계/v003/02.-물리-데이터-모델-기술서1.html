2. 물리 데이터 모델 기술서1

0. 공통 표준 정의 (Common Standards)

0.1. 공통 Audit 컬럼 (Audit Columns)

아래 4개 컬럼은 모든 테이블에 기본적으로 포함됩니다.

컬럼명 데이터 타입 Null Default 설명
created_at DATETIME N now() 데이터 생성 시각
updated_at DATETIME N now() 데이터 최종 수정 시각 (On Update)
created_by BIGINT UNSIGNED N 생성자 ID (System=1)
updated_by BIGINT UNSIGNED N 수정자 ID (System=1)

0.2. 삭제 정책 (Soft Delete)

  • deleted_at (DATETIME, Nullable) 컬럼이 NULL이면 활성, 값이 있으면 삭제된 상태입니다.

1. 유저 및 권한 (Users & Permissions)

1.1. users (사용자)

컬럼명 데이터 타입 Null Key Default 설명
user_id BIGINT UNSIGNED N PK AI 사용자 고유 ID
file_permission_id BIGINT UNSIGNED N FK 루트 파일 권한 ID
inviter_id BIGINT UNSIGNED Y FK 초대자 ID (Self Ref)
login_id VARCHAR(255) N UQ 로그인 아이디
password VARCHAR(255) N 비밀번호 해시 (Bcrypt/Argon2 대응)
email VARCHAR(50) N UQ 이메일
deleted_at DATETIME Y 비활성화 시기

1.2. user_permissions (사용자 권한 상세)

컬럼명 데이터 타입 Null Key Default 설명
user_id BIGINT UNSIGNED N PK, FK users 테이블과 1:1
can_invite_person BOOLEAN N false 사용자 초대 권한
can_share_file BOOLEAN N false 파일 공유 권한
can_store_personal BOOLEAN N false 개인 저장소 사용 권한
can_create_group BOOLEAN N false 그룹 생성 권한

2. 파일 권한 및 용량 (File Permissions & Capacity)

2.1. file_permission_keys (파일 권한 키)

컬럼명 데이터 타입 Null Key Default 설명
file_permission_id BIGINT UNSIGNED N PK AI 권한 고유 ID
parent_permission_id BIGINT UNSIGNED Y FK 상위 권한 키
owner_type VARCHAR(10) N 소유자 타입 ('USER', 'GROUP')
total_capacity BIGINT N 0 총 할당 용량 (Byte)
available_capacity BIGINT N 0 현재 사용 가능 용량 (Byte)
deleted_at DATETIME Y 삭제 시기

2.2. capacity_allocations (용량 할당 이력)

컬럼명 데이터 타입 Null Key Default 설명
allocation_id BIGINT UNSIGNED N PK AI 할당 고유 ID
granter_allocation_id BIGINT UNSIGNED Y FK 용량 원천 할당 ID
receiver_permission_id BIGINT UNSIGNED N FK 용량 받는 키
giver_permission_id BIGINT UNSIGNED Y FK 용량 주는 키 (System=NULL)
allocated_size BIGINT N 할당 용량 (Byte)
expiration_date DATETIME Y 만료일
allocation_type VARCHAR(20) N 타입 ('EVENT', 'GRANT', 'SYSTEM')
description VARCHAR(100) Y 상세 설명
deleted_at DATETIME Y 삭제 시기

3. 파일 시스템 (Virtual File System)

3.1. virtual_directories (가상 디렉터리)

컬럼명 데이터 타입 Null Key Default 설명
directory_id BIGINT UNSIGNED N PK AI 디렉터리 고유 ID
file_permission_id BIGINT UNSIGNED N FK 소속 권한 키
parent_directory_id BIGINT UNSIGNED Y FK 부모 디렉터리 (Root=NULL)
read_level INT N 0 읽기 레벨
write_level INT N 0 쓰기 레벨
name VARCHAR(100) N 디렉터리 이름
depth_level INT N 0 깊이 (Root=0)
deleted_at DATETIME Y 삭제 시기

3.2. virtual_directory_stats (디렉터리 통계)

컬럼명 데이터 타입 Null Key Default 설명
directory_id BIGINT UNSIGNED N PK, FK 디렉터리 ID
total_size BIGINT N 0 하위 포함 총 용량
directory_count INT N 0 하위 디렉터리 개수
file_count INT N 0 하위 파일 개수

3.3. virtual_files (가상 파일)

컬럼명 데이터 타입 Null Key Default 설명
virtual_file_id BIGINT UNSIGNED N PK AI 가상 파일 ID
directory_id BIGINT UNSIGNED N FK 소속 디렉터리
real_file_id BIGINT UNSIGNED N FK 실제 파일 메타데이터
read_level INT N 0 읽기 레벨
write_level INT N 0 쓰기 레벨
name VARCHAR(100) N 파일 이름
extension VARCHAR(50) Y 파일 확장자
deleted_at DATETIME Y 삭제 시기

3.4. real_files (실제 파일)

컬럼명 데이터 타입 Null Key Default 설명
real_file_id BIGINT UNSIGNED N PK AI 실제 파일 ID
storage_uuid VARCHAR(32) N UQ 저장된 파일명 (UUID)
file_hash CHAR(64) N SHA-256 Hash (중복제거)
file_size BIGINT N 파일 크기
storage_path VARCHAR(255) N 저장 위치
storage_type VARCHAR(10) N 타입 ('LOCAL', 'S3')
reference_count INT N 0 참조 카운트
deleted_at DATETIME Y 삭제 시기

4. 업로드 세션 (Uploads)

4.1. upload_sessions (업로드 세션)

컬럼명 데이터 타입 Null Key Default 설명
session_id BIGINT UNSIGNED N PK AI 세션 ID
file_permission_id BIGINT UNSIGNED N FK 업로드 권한 키
user_id BIGINT UNSIGNED N FK 업로더 유저
session_uuid VARCHAR(32) N UQ 세션 UUID
total_parts INT N 총 파츠 수
total_size BIGINT N 총 용량
upload_task_id VARCHAR(255) Y 스토리지 작업 ID
local_path VARCHAR(255) Y 로컬 임시 경로
last_active_at DATETIME N now() 마지막 전송 시각

4.2. upload_parts (업로드 파츠)

컬럼명 데이터 타입 Null Key Default 설명
part_id BIGINT UNSIGNED N PK AI 파츠 ID
session_id BIGINT UNSIGNED N FK 세션 ID
part_number INT N 순서
part_size INT N 용량
etag VARCHAR(100) Y 증명서(ETag)

5. 공유 (Sharing)

5.1. share_links (공유 링크)

컬럼명 데이터 타입 Null Key Default 설명
share_link_id BIGINT UNSIGNED N PK AI 공유 링크 ID
user_id BIGINT UNSIGNED N FK 생성자
share_uuid VARCHAR(32) N UQ 공유 UUID
link_type VARCHAR(15) N 타입 ('INVITE' 등)
expiration_date DATETIME N 만료일
password VARCHAR(255) Y 접근 비밀번호 해시
max_use_count INT N 최대 사용 횟수
current_use_count INT N 0 현재 사용 횟수
name VARCHAR(50) N 공유명
deleted_at DATETIME Y 삭제 시기

5.2. share_directories (공유 디렉터리)

컬럼명 데이터 타입 Null Key Default 설명
share_directory_id BIGINT UNSIGNED N PK AI 공유 디렉터리 ID
share_link_id BIGINT UNSIGNED N FK 공유 링크 ID
parent_directory_id BIGINT UNSIGNED Y FK 부모 디렉터리
name VARCHAR(100) N 이름
depth_level INT N 0 깊이
deleted_at DATETIME Y 삭제 시기

5.3. share_files (공유 파일)

컬럼명 데이터 타입 Null Key Default 설명
share_file_id BIGINT UNSIGNED N PK AI 공유 파일 ID
share_directory_id BIGINT UNSIGNED N FK 소속 디렉터리
virtual_file_id BIGINT UNSIGNED N FK 가상 파일
name VARCHAR(100) N 이름
extension VARCHAR(50) Y 확장자
deleted_at DATETIME Y 삭제 시기

6. 그룹 (Groups)

6.1. group_invites (그룹 초대)

컬럼명 데이터 타입 Null Key Default 설명
share_link_id BIGINT UNSIGNED N PK, FK 공유 링크 ID
group_id BIGINT UNSIGNED N FK 그룹 ID

6.2. groups (그룹)

컬럼명 데이터 타입 Null Key Default 설명
group_id BIGINT UNSIGNED N PK AI 그룹 ID
file_permission_id BIGINT UNSIGNED N FK 그룹 파일 권한 키
owner_id BIGINT UNSIGNED N FK 소유자
name VARCHAR(50) N 그룹 이름
max_member_count INT N 최대 인원
current_member_count INT N 0 현재 인원
deleted_at DATETIME Y 삭제 시기

6.3. group_users (그룹 멤버)

컬럼명 데이터 타입 Null Key Default 설명
group_user_id BIGINT UNSIGNED N PK AI 멤버 ID
group_id BIGINT UNSIGNED N FK 그룹
user_id BIGINT UNSIGNED N FK 유저
read_level INT N 0 읽기 레벨
write_level INT N 0 쓰기 레벨
deleted_at DATETIME Y 삭제 시기

6.4. group_user_permissions (그룹 멤버 권한)

컬럼명 데이터 타입 Null Key Default 설명
group_user_id BIGINT UNSIGNED N PK, FK 멤버 ID
can_manage_user BOOLEAN N false 유저 관리 권한
can_manage_file BOOLEAN N false 파일 관리 권한

7.1. batch_job_queues (배치 큐)

컬럼명 데이터 타입 Null Key Default 설명
batch_job_id BIGINT UNSIGNED N PK AI 작업 ID
job_type VARCHAR(50) N 작업 타입
target_table VARCHAR(64) N 대상 테이블
target_id BIGINT UNSIGNED N 대상 PK
status VARCHAR(15) N 'WAIT' 상태
job_data JSON N 파라미터(JSON)
retry_count INT N 0 재시도 횟수
started_at DATETIME Y 시작 시각
finished_at DATETIME Y 종료 시각
